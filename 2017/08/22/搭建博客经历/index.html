搭建博客经历

2017-08-22

我的博客从搭建到上线，一共用了2天，是用的hexo+github方法搭建的，网上教程很多。个人感觉就算是小白也可以搭建出自己想要的博客，在此分享一些我的经验。


一、前言：

Hexo是基于Node.js的一个快速简洁高效的博客框架，我的博客是用Hexo框架+github免费站点托管服务，在搭建博客之前要准备几个相关工作：
1，下载并安装node.js软件；
2，下载并安装git工具；
3，一个github账号；
4，如果你想个性化域名，还需要购买域名。

二、安装node.js和git：

在各自的官网下载，然后基本一直点下去就安装好了，如果不想装在C盘，在安装过程中改下路径就行了。然后检查这2个软件是否安装成功，使用鼠标点击桌面任务栏的开始按钮,在“搜索程序和文件”的输入框中输入cmd,回车即可进入命令行。接下来在小黑窗口依次输入：git version 按回车，node -v 按回车，npm -v 按回车。如果都能出现对应的版本号，那么就是安装成功！

三、申请github账号：

进入官网 https://github.com/ 注册即可，然后建立仓库，如下图： 仓库名字格式必须是：XXX.github.io，其他选默认就行了，最后点击create repository即可。仓库创建完成后就在你电脑的任意盘下新建文件夹，文件夹名字就取刚刚创建的仓库名。

四、使用npm安装Hexo：

回到你安装git的文件夹，找到git-bash.exe双击进入，然后用cd命令进入XXX.github.io文件夹(例如文件夹建立在d盘下，就输入：cd d:XXX.github.io即可)，然后依次输入：
$ npm install -g hexo-cli
$ hexo init
$ hexo generate
这时目录下就产生了新的文件了，如果你想本地预览一下Hexo博客，可以回到git-bash执行：$ hexo server，接着打开http://localhost:4000 就可以看到效果了。
嫌弃默认主题不好看的话，可以到hexo官网下载喜欢的主题到/themes文件夹下，并打开根目录的_config.yml文件修改theme值，theme: landscape改成theme: 下载的主题名，(注意theme冒号后面必须为一个空格)，然后刷新http://localhost:4000 链接即可。

五、把博客放在github托管：

首先要配置SSH Key，先检查电脑是否有SSH keys，在git-bash执行：
$ cd ~/.ssh，如果提示：No such file or directory，说明你是第一次使用git。
这时需要生成新的SSH Key，在git-bash执行：
$ ssh-keygen -t rsa -C “邮件地址@youremail.com”，如下：按enter后系统会提示要你输入密码：
Enter passphrase (empty for no passphrase):
Enter same passphrase again:
在回车中会提示你输入一个密码，这个密码会在你提交项目时使用，如果为空的话提交项目时则不用输入。这个设置是防止别人往你的项目里提交内容。输入密码一直按enter就成功设置ssh key了。
接下来添加 SSH Key 到 GitHub：1、打开本地C:\Users\Administrator\
.ssh\id_rsa.pub，并复制所有值；2、登陆github系统。点击右上角的 Account Settings—>SSH Public keys —> add another public keys3、把你本地生成的密钥复制到里面(key文本框中)， 点击 add key 就ok了。
可以在git-bash输入下面的命令，看看设置是否成功：
$ ssh -T git@github.com，如果是下面的反馈：
The authenticity of host ‘github.com (207.97.227.239)’ can’t be established.
RSA key fingerprint is
16:27:ac:a5:76:28:2d:36:
63:1b:56:4d:eb:df:a6:48.
Are you sure you want to continue connecting (yes/no)?
输入yes就好，然后会看到：
Hi xxx! You’ve successfully authenticated, but GitHub does not provide shell access.
此时SSH Key 配置成功。

六、实现在线预览：

需要进行最后一步设置，打开根目录的_config.yml文件，在最下面设置deploy，如下：回到git-bash依次执行：
$ hexo generat
$ hexo deploy
完成以上步骤，在浏览器中输入xxx.github.io，即可看到了你博客。

七、个性化域名：

如果嫌弃自己的博客域名不够个性化，没关系，以下几步即可实现。
1，去万网、godaddy等域名注册商购买域名。(这里就不详细说了)
2，域名购买完成后，需要进行域名解析，首先打开git-bash执行：$ ping xxx.github.io此时得到了类似000.000.00.000的ip地址。然后回到注册商网站，在域名控制台进行域名解析，按下图设置，A记录值为得到的ip地址。
3，在根目录的/source文件夹新建CNAME文件，里面只写你购买的域名：www.xxx.xx。
完成后重新布局一下$ hexo generat ；$ hexo deploy 。然后打开网页www.xxx.xx吧。
